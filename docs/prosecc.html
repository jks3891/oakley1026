Purpose

기존 홈페이지에서 불편했던 UI/UX 요소들을 개선하고, 브랜드 아이덴티티와 방향성을 명확히 전달할 수 있도록 리뉴얼했습니다. 불필요한 요소를 과감히 삭제하여 사용자가 직관적으로 브랜드와 상품에 집중할 수 있게 하였습니다.

Target

- 패션에 대해 관심이 많은 사람

- 온라인으로 스포츠 용품을 구매하려는 소비자

Concept

시각적인 요소에 재미를 주기 위해 메인 페이지에 스크롤 애니메이션과 다양한 레이아웃 배치를 주었고, 스포티하고 과감한 비디오와 이미지와의 조화를 맞추기 위해 무채색의 비교적 차분한 컬러를 사용했습니다.

Skill

웹 디자인 트랜드에 맞춰 테일윈드와 제이쿼리 기술을 사용해 코드의 길이를 줄이고, 가독성을 높혔습니다.

AOS 스크롤 애니메이션을 활용해 웹 사이트를 동적으로 만들어서 시각적 흥미를 주었고, 탭 메뉴와 스와이퍼 라이브러리를 활용하여 사용자의 편리성을 더했습니다.

상품 페이지에서 css 변수를 이용하여 반응형 그리드를 만들고, 버튼 토글을 통해 상품의 배열을 조정할 수 있도록 하였습니다.

Main Page

Header

hover, 스크롤 시 애니메이션 효과를 주어 가시성에 중점을 두었고, 반응형 시 사이드바를 배치하여 접근성을 높혔습니다.
- 로고 클릭 시 메인 페이지로 이동합니다.
- 메뉴 hover 시 2차, 3차 메가메뉴가 나오고 클릭 시 해당 페이지로 이동합니다.
- 돋보기 아이콘 클릭 시 검색창이 Slide Down 됩니다.
- 모바일 화면 햄버거 메뉴 클릭 시 사이드바가 나옵니다.

Main-1

비디오 슬라이드를 사용해 사용자의 시각적 흥미를 유도하고, 신상품과 최신 브랜드 이슈를 한눈에 볼 수 있게 했습니다.

Main-2

제품의 장점을 강조하고, 버튼 클릭을 통해 상품 페이지로 바로 이동할 수 있게 하여 소비자의 소비 욕구를 높히도록 구현해 보았습니다.
- 버튼 클릭 시 각 상품 페이지로 이동합니다.

Main-3

버튼 클릭을 클릭하면 팝업 형식으로 나오는 비디오를 통해 해당 브랜드의 기술을 보여주었습니다.

Main-4

탭 메뉴, 슬라이드를 접목시켜 제품을 한 눈에 볼 수 있게 했습니다.
- 버튼 클릭 시 커스텀 페이지로 이동합니다.

Help Button

모든 페이지 우측에 도움 버튼을 고정시켜 사용자의 편리성을 높혔습니다.
- 버튼 클릭 시 도움 관련 정보가 팝업 형식으로 나타납니다.

Main-5

색다른 레이아웃 배치와, 애니메이션을 사용해 신제품 컬렉션을 홍보하는 배너로 시각적 흥미와 사용자의 집중도를 올렸습니다.

Main-6

애니메이션으로 반짝이는 별을 표현해 보았고, 셀럽과 콜라보한 한정판 제품에 대한 정보를 넣었습니다.
- 버튼 클릭 시 상품 상세 페이지로 이동합니다.

Footer

로고에 애니메이션을 추가하여 브랜드를 한번 더 강조했습니다.

Page Up Button

모든 페이지 우측 하단에 버튼을 고정시켜서 클릭 한 번으로 페이지 최상단으로 이동 할 수 있게 하여 사용자의 편리성을 높혔습니다.

Product Slide

hover 시 슬라이드를 구현해 제품의 전, 측, 후면의 이미지를 한 눈에 볼 수 있게 구현 하므로써 사용자의 편리성을 높혔습니다.
- 이미지, 텍스트 클릭 시 상품 상세 페이지로 이동합니다.

Sub Page - 커스텀 페이지
